## 项目介绍

这是一个用于学习node 所写的仿网页微信的即时聊天的<span style='color:red'>客户端</span><br>
项目是用vue + socket-io写的。

![界面示例](https://magiczhuzhu.club/ci/upload/chatroom_main.png)
###### 安装依赖

```bash
npm install
```
###### 启动项目

```bash
npm start
```
###### 打包&&构建

注意在vue.config.js中修改 服务器路径
```bash
npm run build
```

### 功能点

* 聊天广场 
* 点对点聊天 
* 好友功能
* 聊天记录本地存储
